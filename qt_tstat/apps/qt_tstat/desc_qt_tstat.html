Thermostat Demo

This example demonstrates a simple GUI to implement thermostat feature

This application host a server that can interact with the GUI but connecting to the network. After connecting to the network and pressing start on the settings page, type in the address of the EVM followed by port 8081, i.e. http://xxx.xxx.xxx.xxx:8081/ to control the application remotely

NOTE: The ip address of the EVM can be found by either using the "ifconfig" command on the serial console, or by running the Settings -> System Info application in Matrix
